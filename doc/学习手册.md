# 消息增强器单机版 - 学习手册

## 项目包架构

```
cn.redcoral.messageplus
|———— config        // 配置包
|———— constant      // 常量包
|———— entity        // 实体类包
|———— exteriorUtils // 第三方工具包
|———— handler       // 消息处理包
|———— initialize    // 初始化包
|———— interceptor   // 消息拦截器
|———— manage        // 消息管理包
|———— port          // 开发者接口包
|———— properties    // 配置文件包
|———— utils         // 工具包
|———— EnableMessagePlus.java // 启动类注解
```
